# Simulations et notebooks Jupyter — Agrégation SII

Notebooks Jupyter utilisés dans le cadre de la préparation au concours de l'agrégation externe en **Sciences Industrielles de l'Ingénieur (SII)**.

Chaque notebook est exécutable localement ou directement dans le navigateur via [Basthon](https://notebook.basthon.fr).

---

## Traitement du signal

- `fourier_transformees.ipynb` — Les quatre transformées de Fourier (TF, SF, TFTD, TFD), règle Discret↔Périodique, aliasing, fuite spectrale
- `welch_explication.ipynb` — Méthode de Welch, DSP pas à pas : résolution fréquentielle, fenêtrage, moyennage

## Problématique du dossier industriel

- `dilemme_choc_vibration.ipynb` — Système masse-ressort-amortisseur 1 DDL, transmissibilité, réponse au choc, optimisation f₀ et ζ (SAVER 3X90 / ASTM D4728)

## Autres

- `dimensionnement.ipynb` — Dimensionnement
- `pvlib_exemple.ipynb` — Simulation photovoltaïque avec pvlib
